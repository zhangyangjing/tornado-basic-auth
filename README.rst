tornado-basic-auth
=================

.. class:: no-web no-pdf


basic authentication for the Tornado_ web framework.


Installation
------------

The latest stable version of tornado-basic-auth can be installed from pypi:

.. code-block:: bash

  $ pip install tornado-basic-auth
